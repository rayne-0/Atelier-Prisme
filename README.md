# Atelier Prisme — High-End Digital Production

Atelier Prisme is a premium creative studio specializing in digitally produced visuals for modern fashion and fine jewelry brands. We replace the artisanal constraints and logistical overhead of traditional physical photography with a hyper-efficient, editorial-grade digital pipeline.

![Atelier Prisme Monogram](public/assets/ap-monogram-nobg.png)

## 💎 The Vision

Modern brands demand visual excellence at a pace the physical world cannot sustain. Atelier Prisme exists at the intersection of high-end fashion editorial and cutting-edge digital creation. We deliver archive-quality imagery with absolute control over lighting, materials, and environment.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Email**: [Resend](https://resend.com/) for secure lead delivery
- **Deployment**: Optimized for Vercel

## ✨ Key Features

- **Hyper-Editorial Hero**: A cinematic, high-impact landing experience.
- **Dynamic Portfolio**: A filterable, masonry-style archive of fashion and jewelry work.
- **Process Visualization**: A clear 4-step workflow from strategy to final delivery.
- **Conversion Focused**: Integrated booking forms and high-intent CTAs throughout.
- **Glassmorphism Design**: A custom design system reflecting the "Prisme" identity.

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/rayne-0/Atelier-Prisme.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env.local` file with your Resend API Key:
   ```bash
   RESEND_API_KEY=re_your_key_here
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```
